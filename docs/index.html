01. Stock Dashboard

This is a web application I built using React, Bootstrap, and ChartJS. Users can enter the tickers of their favorite stocks and be presented with charts representing the comapany's stock price action and financial performance. The latest news pertaining to the stock/company is also displayed as hyperlinks, so that the user is easily directed to the news source. Furthermore, the user can perform financial analysis on the company using the discounted-cash-flow (DCF) model, with the aid of the information presented, as well as their own. The application uses Fetch API to retrieve the relevant data from a third-party API provider. The ticker banner at the top is a third-party React component modified to fit the needs of this project. The source code for this project can be found here. You can also try the app yourself here.

02. Song Library

This is a web application built on the MERN stack (MongoDB, Express, React, Node.js) that helps users maintain their own personal library of their favorite songs. The front-end features a token-based user authentication with password hashing for access controls and security. Each user can create, retrieve, update, and delete their songs to/from the database, along with filtering their search by a specific artist or genre. This is accomplished through communication with the back-end, which features REST APIs on an Express server connected to the MongoDB database. The source code can be found here.

03. Hotel Property Management Application

This is a desktop application that was built in my Software Design class in university using Java, Java Swing, and MySQL in a small group.
