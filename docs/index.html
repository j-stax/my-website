01. Song Library

This is a web application built on the MERN (MongoDB, Express, React, Node.js) stack that helps users maintain their own personal library of their favorite songs. The front-end features a token-based user authentication combined with password hashing for access controls and security, respectively. Each user can create, retrieve, update, and delete their songs to/from the database, along with filtering their search by a specific artist or genre. This is accomplished through communication with the back-end, which features REST APIs on an Express server connected to the MongoDB database. The source code can be found here.
